QA Automation Project – Selenium WebDriver (Python)

Project Description
This project contains automated UI tests for the website https://www.qa-practice.com/. The goal of the project was to practice test automation of different UI elements using Playwright and Python.

Scope
Tested various UI components, including:
- Input fields
- Select elements
- Checkboxes
- Buttons
- Pop-ups and Alerts
- Other interactive elements
  
Tech Stack:
- Python
- Selenium WebDriver
- Pytest

Features:
- Implemented Page Object Model (POM) for better structure and maintainability
- Used test parametrization to increase test coverage
- Covered both positive and negative test scenarios
- Organized tests using markers
- Bug Findings
  
During testing, several defects were identified:

- 3 issues related to mismatch with requirements
- 1 critical issue where not all entered data was displayed correctly
  
Purpose:
The main goal of the project was to gain hands-on experience in UI test automation and working with different web elements.

How to Run Tests:
- Install dependencies
- Run tests using pytest
